Pyslvs

基本介紹

pyslvs是一套利用 Python 3 與 PyQt5 建立的平面機構模擬與合成系統機構模擬

下載連結

http://www.pyslvs.com/content/%E4%B8%8B%E8%BC%89.html

基本操作

開啟pyslvs可以點選File 再點選Example 可以開啟簡單的範例來練習。

在介面點選右鍵可以建立Point座標來設計聯桿機構運動過程‧。

如果完成構想圖後，可以點選上方option / get expression 或者 python Script轉成你所需的格式。
